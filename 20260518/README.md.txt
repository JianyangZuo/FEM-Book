# 三维杆单元刚度矩阵与应力计算程序

## 1. 程序功能

本程序用于计算三维杆单元 / 空间桁架单元的：

- 单元长度 L
- 方向余弦 cx, cy, cz
- 全局坐标系下的 6 x 6 单元刚度矩阵 Ke
- 单元轴向应变 epsilon
- 单元轴向应力 sigma
- 单元轴力 N
- 单元节点内力 Fe
- 刚度矩阵对称性检查
- 刚度矩阵半正定性检查
- 刚度矩阵奇异性检查
- 刚体平移位移检查
- 刚度矩阵物理意义验证
- 退化单元检查

## 2. 使用单位

程序采用国际单位制：

- 力：N
- 长度：m
- 应力 / 弹性模量：Pa
- 面积：m^2
